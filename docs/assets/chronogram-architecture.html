Chronogram Architecture

Memory Operating System for Agent Ecosystems — Local-First · Hybrid Recall · Adaptive Compaction

Memory Types

  • • Working Memory → Redis (low latency, fast eviction)
  • • Episodic Memory → Neo4j (temporal graph, Graphiti bridge)
  • • Semantic Memory → Weaviate (vectors, hybrid recall)
  • • Procedural Memory → Mixed (graph + semantic, tool traces)

Recall Strategy

  • • Semantic search → conceptual similarity
  • • Lexical matching → exact terms & identifiers
  • • Graph traversal → entities, relations, temporal context
  • • Metadata filters → scope, recency, trust, type
  • • Reranking → bge-reranker-base final pass

Lifecycle Management

  • • Context pressure monitoring (~70% trigger)
  • • Adaptive compaction → derive episodic, semantic, procedural
  • • Salience reinforcement → feedback-driven promotion
  • • Open-loop extraction → preserve unresolved work
  • • Background workflows via Temporal tasks
